## 本集主軸：一條工作流，四個能力
> **讀進來 → 處理 → 抓資料 → 產出去**

| | 能力 | 一句話 | 老師現場 demo |
|---|------|--------|---------------|
| ① 讀 | 讀寫本地檔案 | 它看得到、改得動你硬碟裡的檔案 | 整理凌亂資料夾 + 批次改檔名 |
| ② 處理 | 執行程式與裝工具 | 它能寫程式、跑程式、裝工具 | 寫一段 Python 幫你算全班平均 |
| ③ 抓 | 上網與下載 | 它能自己上網查、自己下載 | 上網抓最新資料／下載素材 |
| ④ 產出 | 生成各種格式檔案 | 它能直接吐出成品檔 | 一句話產出 Word 考卷／Excel 成績表 |

## 開場：為什麼這集很關鍵？（3 分鐘）
- 回扣 EP01：生成式 AI（ChatGPT 網頁版）只能「跟你聊、給你文字」，你要自己複製貼上、自己存檔、自己整理。
- Agent 不一樣：它有「**手腳**」——能直接讀你電腦的檔案、能跑程式、能上網、能把成品存成檔案給你。
- 一句話定調：**「生成式 AI 動嘴，Agent 動手。」**

## 能力 ①：讀寫本地檔案（讀）
### 現場 demo：整理一個凌亂的資料夾
```
這個資料夾很亂，幫我：
1. 依檔案類型分到「講義」「考卷」「圖片」「影片」四個子資料夾
2. 把所有檔名統一改成「日期_主題」的格式
做之前先列一個計畫給我看，我說 OK 再動手。
```

## 能力 ②：執行程式與裝工具（處理）
### 現場 demo：寫一段 Python 幫你算全班成績
```
這份 grades.xlsx 是全班段考成績，幫我：
1. 算出每個人的平均、全班平均、最高最低分
2. 標出不及格的同學
3. 用 Python 跑，把結果存成一份新的 Excel
```

## 能力 ③：上網與下載（抓）
### 現場 demo：上網查資料 / 下載素材
```
上網幫我查「今年的某個時事數據」，整理成 3 個重點，附上資料來源連結。
```

## 能力 ④：生成各種格式檔案（產出）
### 現場 demo：一句話產出一份 Word 考卷
```
幫我出一份國中數學「一元一次方程式」小考：
- 10 題，難度由淺到深，最後 2 題是應用題
- 附詳解
- 直接做成一份 Word 檔，題目卷和答案卷分開
```

## 收尾 1：安全提醒
3 個原則：
1. **半自動為主**：重要動作讓它先規劃、你確認再執行。
2. **限制在專案目錄**：別讓它在整顆硬碟亂跑。
3. **來路不明的工具/Skill 不要亂裝**。
